### 定义临时别名
下列指令可以定义临时别名，终端关闭后失效。
```
$ alias 别名='指令'
```
也可以删除定义的别名。
```
$ unalias 别名
```

### 永久定义别名
通过在配置文件中写入，可永久定义此别名。一般情况下，通过修改'~/.bashrc'，因为我的系统使用了zsh，所以需要修改‘~/.zshrc’。
在文件中添加列语句。
```
alias 别名='指令'
```
